# Krug_Seed_Data

Created by David Mitchell Krug on 16-12-2025
Last Updated: 16-12-2025
Maintained by David Mitchell Krug

## Description

A folder containing experimetnal data from germination experiments and seed mass measurements from trials conducted on three species of 
native seeds from the Yukon, Alberta and British Columbia (Achillea millefolium, Shepherdia canadensis, and Dryas drumondii) as part of David's thesis project looking at germination charachteristics of culturally and ecologically important restoration species from the Yukon. 


*** 

## Table of Contents

|  Name | Description | 
| :---: | :--- |
|  germ_long_metadata | metadata for germination data stored in the file germination_data_combined_longform 
| germination_data_combined_longform | germination data from all germination trials for David's thesis experiments. Data is stored in long format and includes cumulative germination counts. 
| seed_mass_clean | seed mass data for the three trial species. 
| seed_mass_metadata | metadata for seed mass data stored in the seed_mass_clean file.
